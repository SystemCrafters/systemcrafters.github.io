<kbd>dired</kbd></li> <li><kbd>dired-jump</kbd> - open Dired buffer, select the current file</li> <li><kbd>projectile-dired</kbd></li> </ul> <h3><a id="navigation" class="anchor" href="#navigation">¶</a>Navigation</h3><p><strong>Emacs</strong> / <strong>Evil</strong> </p> <ul><li><kbd>n</kbd> / <kbd>j</kbd> - next line</li> <li><kbd>p</kbd> / <kbd>k</kbd> - previous line</li> <li><kbd>j</kbd> / <kbd>J</kbd> - jump to file in buffer</li> <li><kbd>RET</kbd> - select file or directory</li> <li><kbd>^</kbd> - go to parent directory</li> <li><kbd>S-RET</kbd> / <kbd>g O</kbd> - Open file in "other" window</li> <li><kbd>M-RET</kbd> - Show file in other window without focusing (previewing files)</li> <li><kbd>g o</kbd> (<kbd>dired-view-file</kbd>) - Marks a file</li> <li><kbd>u</kbd> - Unmarks a file</li> <li><kbd>U</kbd> - Unmarks all files in buffer</li> <li><kbd>* t</kbd> / <kbd>t</kbd> - Inverts marked files in buffer</li> <li><kbd>% m</kbd> - Mark files in buffer using regular expression</li> <li><kbd>*</kbd> - Lots of other auto-marking functions</li> <li><kbd>k</kbd> / <kbd>K</kbd> - "Kill" marked items (refresh buffer with <kbd>g</kbd> / <kbd>g r</kbd> to get them back)</li> <li>Many operations can be done on a single file if there are no active marks!</li> </ul> <h4><a id="copying-and-renaming-files" class="anchor" href="#copying-and-renaming-files">¶</a>Copying and Renaming files</h4><ul><li><kbd>C</kbd> - Copy marked files (or if no files are marked, the current file)</li> <li>Copying single and multiple files</li> <li><kbd>U</kbd> - Unmark all files in buffer</li> <li><kbd>R</kbd> - Rename marked files, renaming multiple is a move!</li> <li><kbd>% R</kbd> - Rename based on regular expression: <kbd>^test</kbd> , <kbd>old-\&amp;</kbd></li> </ul> <h4><a id="deleting-files" class="anchor" href="#deleting-files">¶</a>Deleting files</h4><ul><li><kbd>D</kbd> - Delete marked file</li> <li><kbd>d</kbd> - Mark file for deletion</li> <li><kbd>x</kbd> - Execute deletion for marks</li> <li><kbd>delete-by-moving-to-trash</kbd> - Move to trash instead of deleting permanently</li> </ul> <h4><a id="creating-and-extracting-archives" class="anchor" href="#creating-and-extracting-archives">¶</a>Creating and extracting archives</h4><ul><li><kbd>Z</kbd> - Compress or uncompress a file or folder to (<kbd>.tar.gz</kbd>)</li> <li><kbd>c</kbd> - Compress selection to a specific file</li> <li><kbd>dired-compress-files-alist</kbd> - Bind compression commands to file extension</li> </ul> <h4><a id="other-common-operations" class="anchor" href="#other-common-operations">¶</a>Other common operations</h4><ul><li><kbd>T</kbd> - Touch (change timestamp)</li> <li><kbd>M</kbd> - Change file mode</li> <li><kbd>O</kbd> - Change file owner</li> <li><kbd>G</kbd> - Change file group</li> <li><kbd>S</kbd> - Create a symbolic link to this file</li> <li><kbd>L</kbd> - Load an Emacs Lisp file into Emacs</li> </ul> <h2><a id="single-dired-buffer" class="anchor" href="#single-dired-buffer">¶</a>Single Dired buffer</h2><p>Closed Dired buffers are just buried!
