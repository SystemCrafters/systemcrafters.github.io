Emacs Has a Built-in Pomodoro Timer?

<div><p><a href="https://youtu.be/JbHE819kVGQ">Watch the video</a> on YouTube! </p> <h2><a id="its-true" class="anchor" href="#its-true">Â¶</a>It's true!</h2><p>Sometimes the best way to focus is to use a short timer like a Pomodoro! </p> <p>You can do this from anywhere in Emacs, just make sure you've loaded Org Mode first. </p> <pre><code class="emacs-lisp"> <span class="org-comment-delimiter">;; </span><span class="org-comment">In your configuration</span> (<span class="org-keyword">require</span> '<span class="org-constant">org</span>) </code></pre> <p>Just use <kbd>M-x</kbd> to run the command <kbd>org-timer-set-timer</kbd> or press <kbd>C-c C-x ;</kbd> inside of an Org file buffer. </p> <p>At the prompt, type in a number of minutes for the timer and press Enter. </p> <p>In the mode line you'll see the timer counting down to show you how much time you have left. </p> <p>If you need to pause the timer run <kbd>org-timer-pause-or-continue</kbd> and to stop the timer, run <kbd>org-timer-stop</kbd>. </p> <p>A Pomodoro wouldn't be complete without a bell, so let's add one! </p> <p>Just set the variable <kbd>org-clock-sound</kbd> to the path of a <kbd>.wav</kbd> file. </p> <pre><code class="emacs-lisp"> <span class="org-comment-delimiter">;; </span><span class="org-comment">Download the sound at https://freesound.org/people/.Andre_Onate/sounds/484665/</span> (<span class="org-keyword">setq</span> org-clock-sound <span class="org-string">"~/Downloads/ding.wav"</span>) </code></pre> <p>Now let's run a 2 second timer to try it out! </p> <p>That's it! Check the link in the description for more info and don't forget to subscribe to System Crafters! Happy Hacking! </p> </div>
